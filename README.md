## About This Project
This project was completed as part of **Task 3 of the Future Interns UI/UX Design Internship**.  
The goal was to design a **collaborative freelancing platform** tailored specifically for **UI/UX designers**, similar to Fiverr or Upwork, with a strong focus on usability, clarity, and real-world UX patterns.

## My Role & Experience
I designed the complete **UI/UX flow** for the WorkHive freelancing website, covering both **desktop and mobile interfaces**.  
The project involved creating structured wireframes, user flows, and interactive prototypes while applying marketplace design principles and industry-standard UX practices.

## Key Features Designed
- Home page with job discovery and designer highlights  
- Job listing and job detail screens  
- Designer profile pages with portfolio sections  
- Proposal submission flow  
- Responsive layouts for desktop and mobile
